PrivateNodeRegistration(privateNodeCertificate: Certificate, gatewayCertificate: Certificate)

Private node registration with its private or public gateway.

When the node is a private endpoint, the gateway must be private. When the node is a private gateway, the gateway must be public.

Parameters

privateNodeCertificate - The certificate of the private node

gatewayCertificate - The certificate of the gateway acting as server

ClientRegistration(clientCertificate: Certificate, serverCertificate: Certificate)

Client registration with a private or public gateway.

When the client is a private endpoint, the server must be a private gateway. When the client is a private gateway, the server must be a public gateway.

Parameters

clientCertificate - The certificate of the private endpoint/gateway

serverCertificate - The certificate of the gateway acting as server
